Event
  ├── subclassOf: (root)
  ├── semantics: “Something that happens in time/space, with optional participants and context”
  ├── core properties (domain = Event)
  │     • eventId [DatatypeProperty] → xsd:string
  │     • descriptionText [DatatypeProperty] → xsd:string
  │     • eventType [DatatypeProperty] → xsd:string         # textual label/code
  │     • startTime [DatatypeProperty] → xsd:dateTime
  │     • endTime [DatatypeProperty] → xsd:dateTime
  │     • startDistanceFromPort_km [DatatypeProperty] → xsd:decimal
  │     • startDistanceFromShore_km [DatatypeProperty] → xsd:decimal
  │     • endDistanceFromPort_km [DatatypeProperty] → xsd:decimal
  │     • endDistanceFromShore_km [DatatypeProperty] → xsd:decimal
  │     • location [ObjectProperty] → Zone                   # where this event occurs
  │     • event bbox geometry [ObjectProperty] → Geometry
  │     • event center point geometry [ObjectProperty] → Geometry
  │     • has participant (membership node) [ObjectProperty] → Group Membership
  │     • validatedBy [DatatypeProperty] → xsd:string        # rule ID / method label
  │     • explainedBy [DatatypeProperty] → xsd:string        # short human explanation
  │     • hasProvenance [ObjectProperty] → Provenance Source
